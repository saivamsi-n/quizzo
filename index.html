About Quizzo


Quizzo is a web application which helps in conducting quiz competetions all the country.


This application helps the teachers in posting the quiz competitions for students. Students can take the test and know the results of the current test and also the other tests they have written in the past. Teachers can also view the results of the students of any test. Leaderboard is present, which displays the top ranker all over the country and also in a particular city.
